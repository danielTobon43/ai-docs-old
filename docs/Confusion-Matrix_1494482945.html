Inteligencia Artificial - documentación central : Confusion Matrix

Created by Daniel Tobon, last modified on Oct 28, 2021

Existen diferentes tipos de matrices de confusion para evaluar el perfomance de un modelo.

  • Confusion Matrix for binary classification

  • Confusion Matrix for multi-class classification

  • Confusion Matrix for multi-label classification

  • Confusion Matrix for object detection

Actualmente, en AITuring se está calculando la matriz de confusión “multi-class classification“ para un modelo de detección de objetos.

El código se encuentra en codecommit:

ebisuu_code/ebisuu3/ebisuu_accuracy/views/matrix_confustion.py

No se pudo determinar en que parte del código se utiliza el modelo para obtener predicciones en el set de imágenes de prueba

No se encontró el cálculo de intersection over union (IOU) para el caso de una matriz de confusión de detección de objetos

Se está utilizando sklearn.metrics import confusion_matrix
