Inteligencia Artificial - documentación central : NVIDIA Inception Program

Created by Daniel Tobon on Nov 24, 2021

Programa de NVIDIA para impulsar startups de inteligencia artificial. AITuring como miembro del programa (AITuringNVIDIA) tiene algunos beneficios que incluyen cursos en Deep learning, accelerated computing y más con certificación, totalmente gratis para los miembros del equipo de Inteligencia Artificial.

Cursos disponibles en NVIDIA Courses.

Thank you for your interest in hands-on training with the NVIDIA Deep Learning Institute (DLI). 

As an NVIDIA Inception member, your startup has complimentary access to DLI hands-on training. The code DLI_INC_OCT21_63_WQEL_66 is suitable for 200 online, self-paced courses for your entire development team. Please note that this code is only valid until September 1, 2022. Any courses you would like to take after that will need a new code requested via the benefits request form on the Inception Member Portal.

Get started anytime, anywhere, on any device with training powered by GPU-accelerated servers in the cloud.  You can earn a certificate of competency to showcase your skills by completing the built-in assessment in courses. View the complete list of training offerings at http://www.nvidia.com/dli Ready to get started with free self-paced courses?Forward this email to your entire development team so they can take courses.  Visit the DLI self-paced course webpage and select the courses you wish to take.Not sure what course to start with? We recommend the following foundational courses:Getting Started with Deep LearningFundamentals of Accelerated Computing with CUDA PythonAccelerating End-to-End Data Science WorkflowsGetting Started with AI on Jetson Nano (free, no code needed)Once you’ve selected a course, click “Enroll Now.”You need to be a member of the NVIDIA Developer program to enroll in a DLI course. If you have a developer account, log in. Otherwise, create a developer account. (Note that the developer account is separate from the NVIDIA Inception Member Portal)Enter the promo code DLI_INC_OCT21_63_WQEL_66 at checkout and click “Apply.”Payment information is not required. Skip this section and click “Continue.”On the next page, click “Submit.”Enjoy your complimentary self-paced training.If you have any questions, please reach out to dli-help@nvidia.com.
