Inteligencia Artificial - documentación central : PASCAL VOC Annotation format

Created by Daniel Tobon, last modified on Nov 19, 2021

Pascal VOC es un archivo XML, a diferencia de COCO, que tiene un archivo JSON. En Pascal VOC creamos un archivo por cada una de las imágenes en el conjunto de datos. En COCO tenemos un solo archivo para el conjunto de datos completo de entrenamiento, prueba y validación. El cuadro delimitador en los formatos de datos Pascal VOC y COCO es diferente.

En AIturing no se utiliza esta estructura para la organización del dataset. En su lugar, se tienen dos carpetas:

  • IMAGENES: Imagenes en formato .jpg

  • LABELS: Marcaciones en formato .xml (PASCAL VOC)

Los cuatro componentes son Anotaciones, ImageSets, JPEGImages y pascal_label_map.pbxt.

JPEGImages es la carpeta que contiene las imágenes originales.

ImageSets proporciona un lista de archivos .txt dentro de la carpeta principal que contiene la lista de imágenes para entrenamiento y validación.

pascal_label_map.pbxt contiene la identificación y el nombre del objeto que se detectará.

item {
 id: 1
 name: ‘cricketers’
}

La última y más importante sección es Annotations, los archivos de anotaciones se crean para cada imagen en la carpeta dada.

Las anotaciones se almacenan en un archivo XML.

Hay diferentes componentes o etiquetas correspondientes en el archivo XML.

  • folder: que contiene imágenes.

  • filename: la ruta relativa de la imagen que está anotada.

  • path: una ruta absoluta del archivo de salida después de las anotaciones.

  • size: alto, ancho en píxeles, la profundidad que indica el número de canales para la profundidad de imagen RGB es 3, para B/N es 1.

  • object: contiene name, pose, truncated, difficult.

  • nombre: contiene el nombre del objeto que se está anotando, en nuestro caso puede ser el nombre de un producto o afiche

  • pose: orientación izquierda, derecha, etc.

  • truncated: si los objetos se extienden más allá del cuadro delimitador, truncated es 1 en caso contrario 0.

  • dificult: si se evalúa como difícil es 1 sino 0.

  • bndbox: cuadro delimitador que consta de la esquina superior izquierda y la esquina inferior derecha (xmin-top left, ymin-top left, xmax-bottom right, ymax-bottom right)

El formato .XML anotaciones de AIturing es distinto al formato estandar PASCAL VOC en algunos puntos.

Se tienen etiquetas adicionales como country, channel, photo_id, image_url, image_name, user_reviewed y labels.

Debido a que la sección object se encuentra internamente en la etiqueta label, es necesario estandarizar este formato si se desea visualizar las marcaciones en alguna herramienta de labeling como las mencionadas en esta sección: Labeling tools
