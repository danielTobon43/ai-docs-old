Inteligencia Artificial - documentación central : 4. Conexión ftp-VM

Created by Daniel Tobon, last modified on Oct 06, 2021

En esta sección se explica el procedimiento de programar en un editor de texto (Sublime Text3 en este caso) en local y ejecución del programa en la instancia VM de GCP. Esto permite que todo el esfuerzo de desarrollo se enfoque en coding sin contar con los recursos físicos necesarios como una GPU.

Se necesita instalar el paquete: sftp en sublime text 3.
