Inteligencia Artificial - documentación central : Object Recognition

Created by Daniel Tobon, last modified on Nov 25, 2021

  • Classification/Recognition: Dada una imagen con un objeto, averigua cuál es ese objeto. En otras palabras, clasifíquelo en una clase de un conjunto de categorías predefinidas.

  • Localization: busque dónde está el objeto y dibuje un cuadro delimitador a su alrededor

  • Object detection: clasifica y detecta todos los objetos de la imagen. Asigne una clase a cada objeto y dibuje un cuadro delimitador a su alrededor.

  • Semantic Segmentation: clasifique cada píxel de la imagen en una clase de acuerdo con su contexto, de modo que cada píxel se asigne a un objeto

  • Instance Segmentation: clasifique cada píxel de la imagen en una clase para que cada píxel se asigne a una instancia diferente de un objeto
