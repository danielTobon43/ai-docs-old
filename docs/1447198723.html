Inteligencia Artificial - documentación central : Creación de paquete base para efficient-det

Created by Camilo H, last modified on Sep 07, 2021

La creación de un paquete independiente separa el mecanismo de ejecución (docker o container preconstruido) de la tarea en sí misma (entrenamiento)

Por lo que se debe crear un paquete que cumpla una estructura similar

  • mi-paquete

    • main.py (Recepción de parámetros)

    • setup.py (Configuración del paquete)

    • orchestrator.py (Ejecuta los comandos/pasos necesarios)

    • carga-de-datos.py

    • entrenamiento.py

    • evaluacion.py

    • Arquitectura de la red (por ejemplo este sería un submódulo de git donde esté el repositorio efficient-det o un fork del mismo

Recepción de parámetros

Los parámetros recibidos para la ejecución se deben capturar de 3 maneras

  1. Parámetros o argumentos en el main.py

    1. python3 main.py --url "URL" --epochs "EPOCHS" --batch "BATCH" --pesos "PESOSSALIDA" --backbone"BACKBONE" --tpu --zone-tpu "ZONE"

  2. Recepción de parámetros desde las variables de entorno

    1. export URL="MY-URL" EPOCHS="NUMEPOCHS" TPU="true"
# Main.py puede acceder al parámetro URL porque es una variable de entorno
python3 main.py

  3. Recepción de parámetros desde un archivo de configuración

    1. # config.env
URL="MY-URL"
EPOCHS="MY-URL"
TPU="true"

# Consola
python3 main.py --config-file config.env
