Created by Daniel Tobon, last modified on Nov 02, 2021

Crear una llave ssh para conexión a bitbucket.

ssh-keygen -t rsa -f ~/.ssh/bitbucket -C daniel_tobon
chmod 400 ~/.ssh/bitbucket

Agregar config file en ~/.ssh/configbitbucket_ssh

Host bitbucket.org
HostName bitbucket.org
IdentityFile ~/.ssh/bitbucket

Agregar Bitbucket repositorio remoto al repositorio en desarrollo

git remote add bitbucket git@bitbucket.org:danielt_aituring/automl_v2.git
