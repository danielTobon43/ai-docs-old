Repositorios - Codecommit

Los repositorios existentes en AWS Codecommit relacionados con el grupo de inteligencia artificial son los siguientes:

  • annotationToStandard

Proyecto en python para estandarizar el formato de anotación (marcación) de AITuring en PASCAL VOC para visualización de labels en labelImg.

  • lookForS3

Proyecto en python para buscar un archivo en todos los buckets de AWS S3.

  • confusionMatrixObjDet

Proyecto en python para calcular la matriz de confusión de un modelo de detección de objetos basado utilizando tfrecords de validación

  • dataToLabelMap

Proyecto para generar un archivo .yaml con la lista de todas las clases marcadas por dataset. Adicionalmente, se puede generar un gráfico .html/.png con el número de clases por bounding box.

  • splitDataset

Proyecto para dividir un dataset en formato AITuring en TRAIN y VALIDACION dado un porcentaje de separación.

  • inspectTFRecord

Proyecto para inspeccionar el formato TFRecord. Convierte el formato TFRecord a texto.

  • aituring_efficientdet

Repositorio para detección de objetos basado en la arquitectura del EfficientDet de google. Esta es la version 1.0 propuesta por AITuring y contiene el pipeline de Machine Learning actual.

  • aituring_dockertraining

Repositorio con la versión keras de aituring_efficientdet.

  • automl

Fork version del código original del Efficientdet de google. Funciona como submodulo en la versión 2 y permite obtener actualizaciones hechas por Google.

  • automl_v2

Repositorio para detección de objetos basado en la arquitectura del EfficientDet de google. Esta es la version 2.0 propuesta por AITuring y contiene el pipeline de Machine Learning actual con metricas de validación, tensorboard, vertex AI, entre otros.

  • api-notraining

Repositorio para detección de afiches y material POP utilizando el oraculo (Feature extraction and matching con Tensorflow). Ver más en Entrega numeral 3 No-Training.pdf

  • aituring_anomalyDetection

Repositorio para la detección de anomalias en una imagen.

  • aituring_imagequality

Repositorio para mejora y pre-procesamiento de una imagen. Permite mejorar la calidad de la imagen.

  • aituring_mllibrary

Repositorio con librerias de machine learning propuestas por antiguos grupos de inteligencia artificial.

  • aituring_base_efficientdet

Repositorio base del efficientdet. Es la misma version del repositorio automl de google.

  • aituring_superResolution

Repositorio para mejora y pre-procesamiento de una imagen. Es similar a la propuesta del repositorio aituring_imagequality

  • aituring_xception

Repositorio para la detección de objetos basados en la arquitectura Xception.

  • aituring_trainembedding

Repositorio para entrenar un embedding para el aprendizaje de una sola vez.

  • aituring_MAQ

Repositorio que tiene como finalidad almacenar los algoritmos del proyecto M.A.Q.

  • aituring_dataYolo

Repositorio para preparar datos para entrenamiento con YOLO

  • sevenEleven_characters

Repositorio para extracción de texto OCR

  • rd_fast_api_inference

Repositorio para inferencia con FAST API

  • efficientDet_bottles

Repositorio con la primeras pruebas del efficientdet
