Inteligencia Artificial - documentación central : Cloud SDK

Created by Daniel Tobon, last modified on Oct 06, 2021

En esta sección se explica la configuración de gcloud; la herramienta de línea de comandos de Cloud SDK.

La CLI de gcloud administra la autenticación, la configuración local, el flujo de trabajo del desarrollador y las interacciones con las API de Google Cloud. Con la herramienta de línea de comandos de gcloud, es fácil realizar muchas tareas comunes en la nube, como crear una instancia de VM de Compute Engine, encender y apagar las máquinas, conexión mediante ssh keys y mucho más.
